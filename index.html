GiuliaFlix: FilMES E SÉRIES

GiuliaFlix é um projeto front-end desenvolvido com foco em design moderno, responsividade e usabilidade, inspirado nas grandes plataformas de streaming como a Netflix. O objetivo foi aplicar de forma prática os conhecimentos em HTML5 e CSS3, criando uma interface intuitiva e atrativa para os usuários.

A estrutura do site foi cuidadosamente planejada. O HTML organiza o conteúdo em seções bem definidas, como o cabeçalho com navegação, o banner de destaque, as listas de filmes por categoria e o rodapé com informações complementares. Já o CSS foi utilizado para estilizar todos os elementos com atenção aos detalhes visuais — cores escuras predominantes, imagens de alta qualidade, botões interativos, tipografia legível e layout adaptável para diferentes tamanhos de tela.

Além do layout responsivo, o projeto inclui um carrossel de filmes, seções temáticas e um destaque principal para lançamentos, simulando a experiência real de uma plataforma de streaming. A navegação é simples e fluida, com menus organizados e elementos visuais que incentivam a interação.

Esse projeto demonstra não apenas minhas habilidades técnicas com HTML e CSS, mas também minha atenção ao design, experiência do usuário e boas práticas de desenvolvimento web. O GiuliaFlix foi um excelente exercício para consolidar conhecimentos e aprimorar a criação de interfaces digitais profissionais.

Criando sua Aventura

Criando Sua Aventura é um projeto desenvolvido com foco em lógica de programação, interatividade e estrutura narrativa, utilizando HTML, CSS e JavaScript. Inspirado nos livros de aventura em que o leitor escolhe o rumo da história, esse projeto foi uma excelente oportunidade para explorar o potencial da programação na criação de experiências envolventes.

O jogo apresenta uma narrativa onde o jogador recebe uma carta misteriosa com pistas sobre uma cidade perdida. A cada cena, o usuário deve tomar decisões que influenciam diretamente o desenrolar da história, podendo alcançar diferentes finais com base nas escolhas feitas.

Detalhes Técnicos O HTML foi utilizado para estruturar o conteúdo do jogo, como o título, a área do texto narrativo e os botões de escolha. O CSS foi aplicado para criar uma aparência imersiva, com cores escuras e tipografia que remete à aventura e mistério. O JavaScript foi responsável por toda a lógica do jogo — controlando a progressão da história, registrando as decisões do jogador e atualizando o conteúdo dinâmico na tela.

Aprendizados Este projeto reforçou conceitos importantes de estrutura de código, funções, condicionais e manipulação do DOM, além de destacar como é possível usar a programação para contar histórias interativas.

AluraPlus

O projeto Alura Plus é uma landing page simples e moderna criada para divulgar o serviço Combo+, que une os cursos da Alura com o Alura Língua em uma única assinatura. A página usa uma estrutura HTML semântica, com seções bem definidas como cabeçalho, conteúdo principal com textos e imagens, botões de chamada para ação e rodapé.

No HTML, as informações são organizadas com títulos claros, descrições e imagens que ilustram o uso do serviço em vários dispositivos. Os botões incentivam o usuário a assinar o serviço, posicionados estrategicamente para facilitar a navegação e conversão.

O CSS do projeto utiliza variáveis para cores e fontes, garantindo um visual consistente e agradável. O layout é feito com Flexbox e Grid, permitindo que os elementos se alinhem e se distribuam de forma organizada. Além disso, o design é responsivo, adaptando-se bem a diferentes tamanhos de tela, de desktops a smartphones. Com essa combinação de HTML semântico e CSS moderno, a página oferece uma experiência visual clara, organizada e acessível, ideal para promover o serviço de forma eficiente e atrativa.
